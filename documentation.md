# Application : mini gestionnaire de tâches sécurisé

Développer une application web permettant :

    - création de compte utilisateur
    - authentification sécurisée
    - gestion CRUD des tâches
    - marquage "terminée / non terminée"
    - filtrage des tâches
    - persistance des données en base relationnelle
    - documentation minimale du déploiement

## Maquettage, structure de l'application et tables relationnelles de la BDD

ils sont réalisés avec Moqups et consultables sur le site sous forme de PNG dans le dossier "schemas", le maquettage représente les 4 pages disponibles dans l'application, la structure réalisée avec Markdown Tree, sépare le back-end du front-end, l'organisation du site sera détaillée plus bas. La BDD comporte 2 tables "user" et "task" avec pour relation le "user_id" de la table "user" dans la table "task".
